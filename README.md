# https-client

[![Build Status](https://travis-ci.org/vmchale/https-client.svg?branch=master)](https://travis-ci.org/vmchale/https-client)

This is a demonstration of how to use [libcurl](https://curl.haxx.se/libcurl/)
to write to a file.

## Demo

Install [atspkg](https://github.com/vmchale/atspkg/tree/master/ats-pkg#readme)
with

```
curl -sSl https://raw.githubusercontent.com/vmchale/atspkg/master/bash/install.sh | bash -s
```

Then:

```
atspkg test
```
